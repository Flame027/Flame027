OSI CMIS Protocol

Introduction

This package consists of a set of smalltalk classes and support files which implement the CMIS (ISO x.710 & x.711) protocol.

Prerequisites

Some interface to a low level transport (such as solistice ACSE or isode) and the ROSE addOn package are required as prerequisites.
In addition, this package requires the 'ASN1 addOn' package, in order to parse cmis GDMO definition files in x.710 format.
