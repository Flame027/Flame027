Overview

The Society of Worldwide Interbank Financial Telecommunication (SWIFT) was formed in 1973 and is headquartered in Brussels. SWIFT operates a worldwide financial messaging network which exchanges messages between banks and financial institutions.

The eXept SWIFT library includes representation classes, parser and generator for SWIFT messages.
This can be used for banking and finance applications, SWIFT data validators, converters and data generators, written in Smalltalk. It is also used in the SWIFT plugin for the eXpecco test automation framework.
