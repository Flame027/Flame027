OSI ROSE Protocol

Introduction

This package consists of a set of smalltalk classes and support files which implement the ROSE (ISO x.219 & x.229) protocol.

Prerequisites

Some interface to a low level transport (such as Solistice ACSE or ISODE) and a corresponding C-interface library are required as prerequisite.
In addition, this package requires the 'ASN1 addOn' package, in order to parse rose service definition files in x.219 format.
