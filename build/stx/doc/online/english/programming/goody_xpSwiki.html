xpswiki

XPSwiki is an open source web tool for eXtreme Programming teams. It supports those XP practices related to requirements gathering and project management - user stories and the planning game.
XPSwiki is based on Swiki, the Squeak Wiki technology. It aims to be an agile and user friendly tool which adapts itself to your development process.

A new XPSwiki version has been released.

Features

XPSwiki is a free and open source tool, written in Squeak Smalltalk. It is based on a wiki, and as such it is quick and simple.

Its planned features include:
  • online tracking of the project, at various levels (user story, iteration, release)
  • process & product metrics
  • a Web Service interface
  • integration with IDEs (Eclipse, NetBeans etc.)
  • integration with SCM tools (CVS, SubVersion etc.)
  • integration with the FIT framework
  • automatic creation of printable reports

Startup

  • In the launcher's settings dialog, select the communication-HTTPServer settings and create/start an http server.
  • Go to the services section, and add a ComancheSwiki Service. Press apply.
  • Select the newly create service, and configure its SwikiRoot (the original xpSqwiki Demo data is provided in "../webServer/data/xpSwiki"), and change the modules class to "Comanche::XPSwikiModule".
  • Press apply.

  • Finally, open a web-browser (mozilla, netscape, opera etc.) on the webServer.
