PVM (Parallel Virtual Machine) Interface Framework

Overview

This package consists of a set of smalltalk classes to interface to the PVM (parallel virtual machine) library.
This allows for parallel operations to be started, monitored and administered on a network of processing hosts. and for ST/X threads to act as worker processes for other hosts.

Documentation

Please refer to the PVM documentation included in the pvm delivery. This software can be obtained via ftp @ netlib2.cs.utk.edu/pub/pvm3. Also, have a look at the newsgroup comp.parallel.pvm (strongly recommended).
