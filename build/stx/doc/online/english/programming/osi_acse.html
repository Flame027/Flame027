OSI Low Level Access (ACSE Interface)

Introduction

This package consists of a set of smalltalk classes and support files to access the Sun Solistice OSI implementation.
OSI Low level access is provided via Smalltalk wrapper classes, which interface to the ACSE (Access Layer) of the Sun product.

Prerequisites

This package requires that the Solistice APLI layer C-library is linked with the Smalltalk executable; this product must be licensed separately from Sun Microsystems.
In addition, the 'ASN1 addOn' and the 'CParser & CType addOn' packages are required as prerequisites.
