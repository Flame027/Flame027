Overview

EDIFACT is a standardized message format for business (B2B) transactions, and used for the electronic transfer of invoices, orders, etc.

The eXept EDIFACT library includes representation classes, parser and generator for EDIFACT messages.
This can be used for B2B applications, EDIFACT data validators, converters and data generators, written in Smalltalk. It is also used in the EDIFACT plugin for the eXpecco test automation framework.
