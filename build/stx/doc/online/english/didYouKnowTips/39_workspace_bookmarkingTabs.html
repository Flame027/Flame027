To add a page to your personalized workspace, and arrange for it to be automatically included whenever a "MyWorkspace" is opened, perform the following steps:
  1. Add the tab ("Add Page" button in the toolbar menu)
  2. Enter the text into the new page
  3. Save the page's contents
    We recommend that you save the contents as a file in the ".smalltalk" directory of your home director (home button in the save-file dialog). It is not a good idea to save it within the Smalltalk installation tree, because that would be lost, whenever you reinstall a new smallalk version.
  4. Add a bookmark, via the "Add Bookmark Here" item in the "Bookmarks"->"My Workspaces" submenu
