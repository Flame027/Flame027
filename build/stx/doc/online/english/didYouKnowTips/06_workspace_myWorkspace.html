If a file named "MyWorkspace.wsp" exists in the Smalltalk startup directory, every opened workspace will automatically add a page on that file's contents.

There is also a workspace menu item called "Add Page with MyWorkspace", which does this for a normal workspace.

This is very useful for commonly reoccurring expressions or text-fragments: simply type them into a Workspaces, and save its contents under the file named "MyWorkspace.wsp".
