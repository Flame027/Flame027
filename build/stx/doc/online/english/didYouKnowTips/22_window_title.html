You can arrange for a prefix string to be added to all of the ST/X windows. This is especially useful, if you have multiple Smalltalk images open on the same screen and you are loosing track of which browser belongs to which image ;-).

To cahnge the window titles, open a settings dialog from the launcher, navigate to the "Window Style" item, enable "Window Label Format" checkbox at the bottom, and enter an appropriate string into the inputfield.

All of your windows get this string as a prefix.
