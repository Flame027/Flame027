Introduction

This source code manager provides access to a Store-database, as used in Cincom VisualWorks Smalltalk systems.

For a general introduction to source code management, please consult the "Programmer's Guide to ST/X - Sourcecode & Revision Management".

Warning

The status of this sourcecode manager is experimental and unmaintained, and chances are high, that it does not work with the most recent Store-DB versions (especially, if Cincom is changing the DB-layout in the process of migrating to Glorp).

Store-DB Settings Dialog

Database Access Parameters

Enter the usual host, user name and password here. This will be used as the default database, for modules for which no per-module information has been entered below.

Per Module Setup

Enter individual, per-module store-DB settings. Each with its own host, user and password. This allows for individual per-module store repositories to be defined.
