Soap Service Settings

Currently the ony setting that can be done here is to enable or disable SOAP services (all or nothing).

The registration/deregistration must (currently) be done programmatically by individual SOAP Services.
See subclasses of SOAPService for details and examples.

It is planned to add a configuration scheme similar to the HTTPService setup scheme in the future.
