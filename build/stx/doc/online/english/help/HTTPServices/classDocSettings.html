Documentation Root Directory

Select the directory which holds the standard Smalltalk/X documentation.
This directory is only required for additional files, such as bitmap images. The ClassDocService will start additional file-services providing access to the pictures-directory found there.

Class List Expiration Time

Specifies the time that a client should cache the data and not send new requests for the class list
This expiration time (change cycle for add or remove classes) is given in days

Class Doc Expiration Time

Specifies the time that a client should cache the data and not send new requests for a classes documentation
This expiration time (change cycle for change the class protocol) is given in days
