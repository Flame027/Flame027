Web Slideshow Service Settings

The Slideshow-service provides access to images and presents them as a slideshow to web clients.

Image Directory

Specifies the (top-) directory which holds the bitmap images.

Allow Slideshow

If on, the slideshow-button is shown, which when clicked presents a self-running slideshow to the browser client. If off, images must be clicked on, to be seen in the client.

Columns and Slides per Page

These two parameters specify how many slides are shown on a page. These slides are small preview versions of the images as present in the slide directory.

Thumbsize

Specifies the size of the thumbnail (i.e. preview) images.

Style, Borders Filenames and ImageSize

These parameters specify how the slides are presented, and which additional information is shown along them.

Allow Uploads

If on, users which known the upload realm password are allowed to upload images to the service directory.

Upload Realm

Specifies the upload realm. That is the userGroup of those which are allowed to upload images. Only used if the "allow uploads" flag is on.
