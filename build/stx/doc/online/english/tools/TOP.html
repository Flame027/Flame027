Programming Tools

General Information

  • Code Editors - Working with the Text and Code Editor

Basic Tools

  • Launcher - the Central Tool for Launching programming tools and showing Smalltalk messages

Programming Tools

  • System Browser - the main tool for programming in Smalltalk
  • New System Browser - the new, improved browser
  • Debugger - the symbolic Smalltalk debugger
  • Changes Browser - a tool for inspecting & recovering the list of user changes
  • Version Diff Browser - compare versions of a class
  • HTML Document Browser - shows the interactive online documentation
  • Workspace - a tool for evaluating Smalltalk expressions
  • Inspector - a tool for inspecting & manipulating Smalltalk objects
  • Methodfinder - to find a method by its desired effect
  • ToDo List - helps to remember things which need to be fixed
  • Breakpoint Browser - find breakpoints quickly
  • SUnit Runner - unit testing framework and testrunner
  • Packager - builds standalone executables and deployable packages
  • Monticello Repository Browser - helps importing code from Squeak and other Smalltalk dialects

User Interface Construction Tools

  • GUI Painter - a tool for building graphical user interfaces
  • Menu Editor - a tool for creating & editing menus for graphical user interfaces
  • Image Editor - a tool for creating & editing images for menus, buttons, etc.
  • Help Tool - a tool for creating & editing help dictionaries for graphical user interfaces
  • Resource Selection Browser - a tool for browsing and selecting resource methods
  • HierarchicalListEditor - a tool to edit hierarchical list resource methods
  • TabListEditor - a tool to edit (notbook-) tab list resource methods
  • TableColumnEditor - a tool to edit table-column resource methods

File Tools

  • File Browser - a tool for viewing & manipulating files and directories (obsolete)
  • New File Browser (V2) - an improved file browser
  • International Language Editor - a tool to manage national language translations
  • XML Inspector - shows the structure of an XML document

Monitoring Tools

  • Process Monitor - shows & manipulates Smalltalk processes
  • Semaphore Monitor - shows semaphores
  • Memory Monitor - shows the overall memory usage of the Smalltalk system
  • Memory Usage - shows the memory usage by class
  • Event Monitor - shows the incoming window events
  • WindowTree View - shows a view''s widget hierarchy
  • Object Module Information - shows loaded DLLs (class libraries)
