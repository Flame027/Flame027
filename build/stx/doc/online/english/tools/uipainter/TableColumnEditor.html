Introduction

The ST/X Table Column Editor allows you to build and maintain table-column definitions as used in a dataSet (DSV) view.

Starting the Table Column Editor

The TabListEditor is typically started as a subApplication from the UI-Painter.

Most of its functionality is similar to other editors - especially the Menu Editor.
